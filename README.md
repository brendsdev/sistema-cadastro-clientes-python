# Sistema de Cadastro de Clientes

Aplicação desktop desenvolvida em Python para cadastro e gerenciamento de clientes.

## Tecnologias
- Python
- CustomTkinter
- OpenPyXL
- Excel como base de dados

## Funcionalidades
- Cadastro de clientes
- Armazenamento automático em planilha Excel
- Limpeza automática do formulário
- Criação automática do banco de dados

## Como executar

git clone https://github.com/brendsdev/sistema-cadastro-clientes-python.git

pip install customtkinter openpyxl

python app.py
